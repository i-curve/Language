# html中的标签
1. 超链接
```
<a href="https://baidu.com">message</a>
target="_black" 会在新的窗口中打开链接,默认本窗口
name: 锚,跳转到页面内固定的id命名位置
```
2. 图像
```
<img src="url"/>
```
3. 视频
```
<audio>
```
